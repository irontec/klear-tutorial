text¶

  • Volver a Tipos de Campos

El campo text es un campo simple de texto.

Configuración¶

fieldName:
  title: _("Título del campo")
  type: text
  prefix: _("prefijo para el campo")
  sufix: _("sufijo para el campo")
  maxLength: 225

Como se puede apreciar, la configuración es bastante básica. Con definir el type es suficiente.
