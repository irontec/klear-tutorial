Models y Mappers¶

Desde la interfaz de Klear, acceder a los datos de nuestras tablas es automática mayormente y solo hacemos uso de códigos en el yaml para obtener, filtrar, decorar y modificar de manera gráfica estos datos. Sin embargo, existe exepciones o casos especiales en el que no todo lo que se necesita está programado y para ello debemos hacer uso de las funciones bases que nos ofrece klear para obtener la informacion deseada.

Un resumen, los models sirven guardar y los mappers para obtener la información deseada de la base de datos (ya sea para mostrar, actualizar o borrar) básicamente.
